# 🎲 お題ゲームメーカー - エルそれ！

data.jsonベースのお題自動生成システムを搭載したPWA（Progressive Web App）アプリケーションです。

## ✨ 機能

- **自動お題生成**: data.jsonから初期値と単語をランダム組み合わせ
- **確率調整**: rare値による出現確率の調整
- **逆回答モード**: 低確率（10%）で発動する特殊モード
- **複数抽選**: 最大10個まで同時抽選可能
- **メモ機能**: 抽選結果にメモを追加
- **個別コピー**: 各お題ごとにSNS投稿用フォーマットでコピー
- **管理画面**: お題データの編集・管理
- **オフライン対応**: PWAでオフラインでも利用可能

## 🚀 使い方

### 基本的な使い方

1. **設定を入力**
   - アプリのタイトル
   - 配信のURL（任意）
   - ハッシュタグ（任意）

2. **お題を抽選**
   - 最大抽選数を設定（1-10）
   - 「抽選（1つ追加）」で1つずつ追加
   - 「一括抽選（満枠まで）」で一気に抽選

3. **結果を活用**
   - 各お題の「コピー」ボタンで個別コピー
   - 必要に応じてメモを追加
   - 「再抽選」で別のお題に変更可能

### お題フォーマット

自動生成されるお題のフォーマット：
```
(ひらがな1文字)から始まる(単語)といえば？
```

例：
- (あ)から始まる(かたいもの)といえば？
- (さ)から始まる(かわいいもの)といえば？

### 逆回答モード

10%の確率で発動し、赤枠で表示されます。

## 🔧 管理画面

お題データ（data.json）を編集できる管理画面を用意しています。

### アクセス方法

1. `admin.html`を開く
2. パスワードを入力（デフォルト: `admin123`）
3. ログイン

### 使い方

1. **お題の追加**: 通常と逆の2つを入力して「追加」
2. **お題の編集**: 各項目を直接編集
3. **お題の削除**: 「削除」ボタンをクリック
4. **保存**: 「保存（ダウンロード）」で`data.json`をダウンロード
5. **反映**: ダウンロードしたファイルでプロジェクトの`data.json`を置き換え

### パスワードの変更

`admin.js`の以下の行を編集してください：
```javascript
const ADMIN_PASSWORD = 'admin123';  // ここを変更
```

### コピーフォーマット

```
【アプリのタイトル】お題「お題内容」を使ってゲーム中！　気になった人はこちら！配信URL #ハッシュタグ
```

## 📱 インストール方法

### Webブラウザから

1. ブラウザでアプリを開く
2. アドレスバーの「インストール」ボタンをクリック
3. ホーム画面に追加される

### ローカルで実行

1. このリポジトリをクローンまたはダウンロード
2. `index.html`をブラウザで開く
3. HTTPSサーバーで公開する場合は以下のコマンドを実行：

```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合（http-serverをインストール）
npx http-server -p 8000
```

## 🎨 アイコン生成

アイコンファイルを生成するには：

1. `generate-icons.html`をブラウザで開く
2. 自動的に`icon-192.png`と`icon-512.png`がダウンロードされる
3. ダウンロードしたファイルをプロジェクトルートに配置

## 💾 データ構造

`data.json`は以下の構造で管理されています：

```json
{
  "initial": [
    { "key": "あ", "rare": 0 },
    { "key": "が", "rare": 1 }
  ],
  "words": [
    { "normal": "かたいもの", "not": "かたくないもの", "rare": 0 }
  ]
}
```

- **initial**: ひらがな1文字のリスト（rare=0: 通常, rare=1: 低確率）
- **words**: お題の単語リスト（normal: 通常, not: 逆, rare: 確率調整）
