# 🎲 お題ゲームメーカー

オリジナルお題を作成して楽しむPWA（Progressive Web App）アプリケーションです。

## ✨ 機能

- **お題管理**: お題を自由に作成・削除
- **重み付き抽選**: 確率を調整してお題を抽選
- **複数抽選**: 最大10個まで同時抽選可能
- **メモ機能**: 抽選結果にメモを追加
- **コピー機能**: SNS投稿用のフォーマットでコピー
- **履歴管理**: 抽選履歴を自動保存
- **オフライン対応**: PWAでオフラインでも利用可能

## 🚀 使い方

### 基本的な使い方

1. **設定を入力**
   - アプリのタイトル
   - 配信のURL（任意）
   - ハッシュタグ（任意）

2. **お題を登録**
   - お題を入力
   - 確率調整（重み）を設定（1-10）
   - 「お題を追加」ボタンをクリック

3. **お題を抽選**
   - 最大抽選数を設定
   - 「お題を抽選する」ボタンをクリック

4. **結果をコピー**
   - 必要に応じてメモを追加
   - 「結果をコピー」ボタンでクリップボードにコピー

### お題フォーマット例

```
(あ)から始まる(食べ物)といえば？
(赤い)なもの、(赤い)なこと
```

### コピーフォーマット

```
【アプリのタイトル】お題「お題内容」を使ってゲーム中！　気になった人はこちら！配信URL #ハッシュタグ
```

## 📱 インストール方法

### Webブラウザから

1. ブラウザでアプリを開く
2. アドレスバーの「インストール」ボタンをクリック
3. ホーム画面に追加される

### ローカルで実行

1. このリポジトリをクローンまたはダウンロード
2. `index.html`をブラウザで開く
3. HTTPSサーバーで公開する場合は以下のコマンドを実行：

```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合（http-serverをインストール）
npx http-server -p 8000
```

## 🎨 アイコン生成

アイコンファイルを生成するには：

1. `generate-icons.html`をブラウザで開く
2. 自動的に`icon-192.png`と`icon-512.png`がダウンロードされる
3. ダウンロードしたファイルをプロジェクトルートに配置

## 💾 データ保存

- すべてのデータはブラウザのLocalStorageに保存されます
- ブラウザのキャッシュをクリアするとデータが失われる可能性があります
- 重要なお題はバックアップをおすすめします
